# SDL-imGui-cmake-starter

An example of using [Dear ImGui](https://github.com/ocornut/imgui) with [SDL](https://www.libsdl.org).

# Install SDL first
```bash
git clone https://github.com/libsdl-org/SDL
cd SDL
mkdir build
cd build
../configure
make
sudo make install
```
